1-> Installed the json server  using  this command in git bash-
npm install -g json-server

2> Created Address_Book.json file  using this command -
nano Address_Book.json

3-> Saved the data in json format.

4>To get the local host ,command used-
json-server --watch Address_Book.json

 5>By searching the above link in browser,we get all the data stored in Address_Book.json
